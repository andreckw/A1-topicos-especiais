# A1-topicos-especiais
 
## Comandos para criação do VENV
``` shell
    py -m venv .venv
    .venv/Scripts/activate
```


## Comandos para o flask
``` shell
    pip install flask
    pip install -U Flask-SQLAlchemy
    pip install -U Flask-WTF
    
    flask --app main run
```
